# Button

A versatile button component with multiple variants and sizes based on shadcn/ui design.

## Installation

```bash
npm install @megazord-ui/ui
```

## Usage

```tsx
import { Button } from '@megazord-ui/ui';

export default function MyComponent() {
  return <Button>Click me</Button>;
}
```

### Variants

- **default**: Primary button style with solid background
- **destructive**: For destructive actions (delete, remove, etc.)
- **outline**: Button with border and transparent background
- **secondary**: Secondary button style
- **ghost**: Transparent button with hover effect
- **link**: Link-styled button with underline

### Sizes

- **default**: Standard button size (h-10)
- **sm**: Small button size (h-9)
- **lg**: Large button size (h-11)
- **icon**: Square button for icons (h-10 w-10)
