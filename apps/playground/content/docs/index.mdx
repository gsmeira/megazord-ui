# Megazord UI

A React UI component library built with TailwindCSS v4 and inspired by shadcn/ui design principles.

## Features

- **Built with TailwindCSS v4**: Modern utility-first CSS framework
- **Customizable theme**: Easy theme customization with CSS variables
- **Tree-shakeable**: Import only what you need
- **TypeScript support**: Full type safety out of the box
- **Tested with Vitest**: Comprehensive test coverage
- **Multiple variants and sizes**: Flexible component configurations

## Installation

```bash
npm install @megazord-ui/ui
```

## Quick Start

Import the styles and components:

```tsx
import '@megazord-ui/ui/styles.css';
import { Button } from '@megazord-ui/ui';

export default function App() {
  return <Button>Click me</Button>;
}
```
